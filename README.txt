

Dependencies:
1. argparse;
2. pandas;
3. numpy;
4. scipy.stats;
5. sklearn.


. Run the script.
	python3.5 EM Algorithm.py [argv1] [argv2]
	* [argv1]: first argument refers to the dataset;
	* [argv2]: second argument refers to the number of clusters; 
	* e.g.  python3.5  EM Algorithm.py Iris.csv 3
	* Any problem about the argument, can review the help.
		python3.5  EM Algorithm.py -h
